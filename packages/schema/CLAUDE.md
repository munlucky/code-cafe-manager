# @codecafe/schema

> YAML/JSON 스키마 검증 (향후 확장 예정)

## Flow

```
[현재 상태: 최소 구현]
        │
        ▼
┌───────────────────────────────────────────────────────────────────────┐
│  src/index.ts                                                         │
│  - 현재 비어있음 (향후 확장 예정)                                        │
└───────────────────────────────────────────────────────────────────────┘
```

## 현재 상태

이 패키지는 향후 중앙화된 스키마 검증을 위해 예약되어 있습니다.
현재 스키마 검증은 `@codecafe/core/schema/`에서 수행됩니다.

## 향후 계획

```
┌───────────────────────────────────────────────────────────────────────┐
│  예상 구조                                                             │
│                                                                       │
│  src/                                                                 │
│  ├── index.ts           # 모든 스키마 export                          │
│  ├── cafe.ts            # Cafe 설정 스키마                            │
│  ├── workflow.ts        # Workflow 정의 스키마                         │
│  ├── role.ts            # Role 정의 스키마                             │
│  ├── terminal.ts        # Terminal 설정 스키마                         │
│  └── provider.ts        # Provider 설정 스키마                         │
└───────────────────────────────────────────────────────────────────────┘
```
