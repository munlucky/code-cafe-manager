  # Desktop Design Migration Plan                                               
                                                                                
  ## Overview                                                                   
                                                                                
  **목표**: 새 디자인                                                           
  프로젝트(`docs/desktop-design-refactor/codecafe-desktop/`)의 스타일을         
  기존 desktop 패키지에 적용

  적용

  ## Phase 1: Foundation (디자인 시스템)                                        
                                                                                
  ### 1.1 tailwind.config.cjs 업데이트                                          
                                                                                
  **파일**: `packages/desktop/tailwind.config.cjs`                              
                                                                                
  **변경 내용**:                                                                
  - `cafe` 컬러 팔레트 추가 (950-100)                                           
  - `brand` 액센트 컬러 추가 (amber 계열)                                       
  - `terminal` 배경색 추가                                                      
  - 폰트 패밀리 추가 (Inter, JetBrains Mono)                                    
  - 기존 색상은 호환성 alias로 새 값 매핑                                       
                                                                                
  ```javascript                                                                 
  colors: {                                                                     
  cafe: { 950: '#0c0a09', 900: '#1c1917', ... },                                
  brand: { DEFAULT: '#d97706', hover: '#b45309', light: '#fbbf24',              
  subtle: '#78350f' },                                                          
  terminal: { bg: '#120f0e' },                                                  
  // Compatibility aliases                                                      
  background: '#0c0a09', card: '#292524', coffee: '#d97706', ...                
  }                                                                             
  ```                                                                           
                                                                                
  ### 1.2 index.html 폰트 로딩 추가                                             
                                                                                
  **파일**: `packages/desktop/src/renderer/index.html`                          
                                                                                
  **변경 내용**:                                                                
  - Google Fonts 링크 추가 (Inter, JetBrains Mono)                              
  - CSP 헤더에 `fonts.googleapis.com`, `fonts.gstatic.com` 허용 추가            
                                                                                
  ### 1.3 index.css 스크롤바 스타일 업데이트                                    
                                                                                
  **파일**: `packages/desktop/src/renderer/styles/index.css`                    
                                                                                
  **변경 내용**:

  - 기존 `scrollbar-custom` 색상을 cafe 팔레트로 업데이트                       
  - body 기본 스타일 추가 (`bg: cafe-950`, `color: cafe-200`)
